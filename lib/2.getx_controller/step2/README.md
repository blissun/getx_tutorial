# Step 2: 고급 상태관리

이 예제는 GetX의 고급 상태관리 기능을 보여줍니다.

## 주요 기능
1. Rx 타입을 사용한 상태관리
2. 사용자 정의 클래스의 상태관리
3. Worker를 사용한 상태 변화 감지

## 코드 설명
- `UserController`: 사용자 정보를 관리하는 컨트롤러
- Rx 타입을 사용한 상태 변수 정의
- Worker를 사용한 상태 변화 감지 및 처리
- 복잡한 객체의 상태관리 예시

## 학습 포인트
1. Rx 타입을 사용한 상태 변수 선언
2. 사용자 정의 클래스의 상태관리 방법
3. Worker를 사용한 상태 변화 감지 방법
4. 복잡한 상태 업데이트 처리
